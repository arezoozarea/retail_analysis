# Customer Retail Analysis

This project analysis customer behavior for a retail dataset (Kaggle) using a complete end to end analytical pipeline.  
The workflow includes descriptive profiling, RFM segmentation, cohort analysis, churn labeling,basket and network analysis, churn prediction, customer level churn scoring and Retention Strategy (Final Output).

 The full visual pipeline is available here:  
`visuals/pipeline/workflow.png`

Notebooks are organized step-by-step in the `/notebooks` folder and can be executed independently.
